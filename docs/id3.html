id3.coffee

¶

Parses ID3v2 tags.

Audio files like MP3s include some metadata about the track, like artist, album, title, year, etc. These are usually prepended to the actual, encoded audio.

Artwork:

Some audio files include artwork (cover, etc.) embedded in the file itself.

Tags:

Currently, only a few tags are read (that's because I'm too lazy to include all of them).

The code for the parser is open sourced on Github under the MIT license.

Converts a buffer slice s to a number. Not using the in-built
Buffer#readUint*() as numbers may use unsynched values where the MSB of a byte
is shifted to the next (higher) byte.
ebb, or "eight-bit-bytes" uses 8 bits/byte when true
             		                            
_toNumber = (s, ebb) ->
  b = if ebb then 8 else 7
  switch s.length
    when 2
      return s[1] | (s[0] << b)
    when 3
      return s[2] | (s[1] << b) | (s[0] << (2 * b))
    when 4
      return s[3] | (s[2] << b) | (s[1] << (2 * b)) | (s[0] << (3 * b))
